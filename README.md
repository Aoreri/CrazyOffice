# 🎮 Sistem Analiz Projesi 

## 📌 Proje Hakkında

## ⚙️ Gereksinimler

Projeye katılmadan önce aşağıdaki araçların kurulu olması gerekmektedir:

- **Unity Versiyonu:** `6000.4.1f1`
- GitHub Desktop

---

## 🚀 Projeye Katılım

1. Proje sahibinden GitHub repository linkini alın
2. Repository’ye erişim izni isteyin
3. GitHub Desktop üzerinden projeyi klonlayın

---

## 📥 Projeyi Bilgisayara Kurma

### 1. GitHub Desktop ile Klonlama

- GitHub Desktop’ı aç
- `File > Clone Repository` seç
- URL sekmesine repo linkini yapıştır
- Local path seç
- `Clone` butonuna bas

---

### 2. Unity Hub ile Açma

- Unity Hub’ı aç
- `Add Project` → `Add from disk`
- Klonladığın klasörü seç
- Unity versiyonunun `6000.4.1f1` olduğundan emin ol

## 🔒 Çalışma Kuralları

- ❌ Direkt `main` branch’e push atmayın
- ✔️ Her zaman yeni branch oluşturun
- ✔️ Yaptığınız değişiklikleri **Pull Request (PR)** ile gönderin
- ✔️ Anlaşılabilir commit mesajları yazın

---

## 📤 Pull Request Süreci

1. Yeni branch oluştur
2. Değişikliklerini yap
3. Commit at
4. GitHub Desktop üzerinden `Push`
5. GitHub’da PR oluştur

---

## ⚠️ Önemli Notlar

- Unity sahnelerinde (Scenes) çakışma olmaması için dikkatli olun
- Aynı dosya üzerinde aynı anda çalışmamaya özen gösterin
- `.gitignore` dosyasını değiştirmeyin
